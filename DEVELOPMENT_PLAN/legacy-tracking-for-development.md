## Ledger Status

This ledger tracks primary development scope discovered after a phase was
previously closed. It does not track compatibility helpers, deprecated paths, or
temporary scaffolds; those remain in
[legacy-tracking-for-deletion.md](legacy-tracking-for-deletion.md).

One development row is currently active. The project is replacing
ROM-dependent Atari demo/canonical examples with a repo-owned, copyright-free
visual discrete-control environment, `KeyDoorGrid-v0`. Phase `8` owns the
environment and default example plumbing; Phase `9` owns the algorithm matrix
and convergence-threshold retargeting.
